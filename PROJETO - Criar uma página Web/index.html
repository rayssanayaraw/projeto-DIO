Estrutura básica do HTML

O <head> funciona como o "cérebro" invisível da sua página, organizando todas as "regras" e metadados que o navegador precisa para entender e carregar o site. Já o <body> é onde a mágica acontece: ele mostra tudo o que o usuário vai ver e interagir, desde textos e imagens até formulários e vídeos.

Com o HTML, você pode construir a estrutura básica de qualquer conteúdo web. É possível escrever textos, usar títulos de diversos tamanhos com as tags <h1> a <h6>, organizar informações em listas numeradas (<ol>) ou com marcadores (<ul>). Além disso, você pode conectar páginas e outros sites através de links clicáveis usando a tag <a>. E tem muito mais que você pode fazer diretamente no HTML para estruturar seu conteúdo!

Formatações

O <div> não tem nenhum significado semântico próprio. Ele é usado principalmente para agrupar outros elementos HTML para fins de estilização (com CSS) ou manipulação (com JavaScript).

Assim como o <div>, o <span> não tem significado semântico próprio. Ele é usado para aplicar estilos a pequenas partes de texto ou a elementos em linha dentro de um parágrafo, por exemplo.

A tag <embed> é usada para incorporar conteúdo externo (como documentos PDF, vídeos, áudios, animações Flash, ou outros tipos de mídia) diretamente no seu documento HTML. É um elemento de "slot" para conteúdo plug-in.
Observação: Embora o <embed> seja funcional, para incorporar vídeos e áudios, as tags <video> e <audio> são preferíveis, pois oferecem mais controle e melhor suporte. Para incorporar conteúdo de outras páginas web, o <iframe> é mais comum (como para incorporar vídeos do YouTube ou mapas do Google).

Tabelas

As tabelas HTML são usadas para organizar dados em linhas e colunas, tornando a informação estruturada e fácil de ler. Elas são ideais para apresentar dados tabulares, como informações financeiras, listas de produtos, horários, ou qualquer conjunto de dados que se beneficie de uma organização em grade.
